Technology

Our servers are running Debian Linux with NGINX and PHP7.1 and a MariaDB database, but custom servers are also available.
By using an up to date version our servers will be secure against most known vulnerabilities.
We make your website faster by using an in-memory cache, for instance Memcached or Redis. This way the server only needs to load the part of the website that has changed and gets the unchanged data from memory.
Frontend caching is also available, for instance with Varnish or with CloudFlare.
We mostly use WordPress as a CMS but it's also possible develop a custom solution for your specific needs.

Security

SQL injection, NAXSI, LetsEncrypt free TLS, Cloudflare Anti-DDoS.
With years of experience in web application security, our applications will be higly secure. We also provide a free TLS/SSL certificate via LetsEncrypt or CloudFlare by default. This makes sure that when someone connects to your website, the connection will be encrypted. This means that no third parties can read or modify the data while it's being sent from the server to the browser.
We can install NAXSI WAF for an extra layer of security. We also offer free DDoS protection through CloudFlare.
